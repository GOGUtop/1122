# 云洞酒馆 iOS v3.4

本版基于 v3.3 增加快捷操作、缩小版酒馆画中画、自动防息屏，并继续保留稳定的画中画完成提示和系统横幅映射。

## 本版变化

- 悬浮球工具面板新增「快速重Roll」按钮。
- 新增「关闭记录」「新建记录」两个快捷按钮，用于快速触发 SillyTavern 页面内对应操作。
- 画中画内容改成更像缩小版酒馆的镜像样式：顶部状态栏、AI 气泡、底部输入状态条。
- 回复完成后，画中画提示会显示建议操作，例如「快速重Roll / 新建记录」。
- 新增「生成时自动防息屏」开关。
- 后台保活增加短时后台任务续跑，继续使用静音音频心跳和服务端实时桥接。

## 使用

上传到 GitHub 后运行 Actions 构建 IPA。服务端插件不用更换。

快捷按钮是通过网页 DOM 自动点击 SillyTavern 原有按钮实现的。如果某个主题或插件改了按钮名称，按钮可能提示“没有找到”。这种情况下网页原按钮仍可正常使用。

## 后台说明

iOS 不能像安卓一样永久后台运行任意网页。本版继续使用静音音频心跳、系统画中画、短时后台任务和服务端实时桥接来增强稳定性。生成中会自动保持屏幕常亮，生成结束后由后台保活状态决定是否恢复。
