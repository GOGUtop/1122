# 云洞酒馆 iOS v4.0

本版重点修复两个卡顿点：

- 键盘收回后立刻再次点击输入框的卡顿。
- 角色卡点开后，聊天记录加载时的卡顿。

## 主要改动

1. 键盘二次点击深度修复
   - 输入框 touch/pointer/focus 前就进入轻量模式。
   - 取消键盘收回后的重型恢复任务。
   - 角色卡图片扫描、网页动画恢复都延后到空闲期。

2. 角色卡 / 聊天记录加载优化
   - 点角色卡时自动进入聊天加载轻量模式。
   - 检测到一次性加载大量聊天消息时自动降负载。
   - 老聊天消息启用 content-visibility，屏幕外消息不提前重绘。

3. 保留功能
   - 快速重 Roll
   - 下载中心与导出接管
   - 选区长截图
   - 画中画文字提示
   - 画中画提示映射横幅
   - 后台保活

服务端插件不用更换。
