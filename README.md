# 云洞酒馆 iOS v3.6

本版目标：丝滑、能下、不卡、稳定。

## 新增 / 修复

- 键盘弹出、收回时自动进入轻量性能模式，减少 WebView、液态 UI 和画中画镜像一起抢性能。
- 画中画网页镜像改为自适应帧率：键盘/滚动时降频，生成中提频，静止时低频刷新。
- 修复 SillyTavern 导出下载：支持普通下载、`blob:` 导出、`data:` 导出。
- 新增下载中心：导出的角色卡、世界书、正则、预设、聊天记录等会进入下载中心，并可保存/分享。
- 导出后默认自动弹出系统保存/分享面板，可在设置中关闭。
- 保留快速重 Roll、画中画、画中画提示映射横幅、长截图、后台保活。

## 构建

覆盖上传 GitHub 后运行 Actions 构建 IPA。版本号：3.6（27）。

## 服务端插件

服务端插件不需要更换。
