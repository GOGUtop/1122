# TavernSwitcher v2.7 DogTavern Tools

本版在 v2.6 的画中画横幅映射基础上，加入 DogTavern-Lite 风格工具：

- 选中 AI 消息文字后，网页内会浮现「划词翻译 / 生成卡片」按钮。
- 「生成卡片」会把选中的 AI 回复做成分享图，并自动弹出系统分享页。
- 「划词翻译」使用 Microsoft Edge 翻译链路，由 iOS 原生请求，避免网页 CORS 拦截。
- 悬浮球工具面板新增「错误翻译」。
- 错误翻译内置 40+ 常见错误码字典，未命中的原文可继续调用 Edge 翻译。
- 悬浮球与展开面板已重新制作，未展开吸边，展开时自动离开边缘。

## 构建

上传到 GitHub 后进入 Actions，运行 Build iOS IPA。生成的 IPA 仍需要签名安装。

## 使用

1. 打开酒馆网页。
2. 长按或拖选 AI 回复文字。
3. 弹出工具后点「划词翻译」或「生成卡片」。
4. 需要翻译报错时，展开悬浮球，点「错误翻译」。

## 注意

Edge 翻译依赖微软公开翻译授权接口。如果该接口被网络、地区或微软策略临时限制，会显示失败原因；内置错误码字典不依赖网络。
