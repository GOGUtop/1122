# 云洞酒馆 iOS v3.5

本版基于 v3.4 做稳定修正：删除不可稳定触发的「关闭记录」「新建记录」，保留并加强「快速重Roll」，同时把画中画改成「网页镜像优先 + 实时文字兜底」。

## 本版变化

- 悬浮球面板只保留稳定快捷操作：快速重Roll、选区截图、画中画、切换云洞、设置、返回、刷新、主页。
- 删除「关闭记录」「新建记录」两个按钮。它们在不同 SillyTavern 主题/插件里 DOM 不统一，容易无效。
- 快速重Roll 增强：优先调用酒馆上下文函数，失败后尝试最后一条 AI 消息的重生/滑动按钮和更多菜单。
- 画中画新增网页镜像模式：优先把当前 WKWebView 可见画面缩小显示到画中画中，像小录屏一样显示酒馆界面。
- 当 iOS 后台冻结网页截图时，仍继续使用服务端实时桥接文字兜底，所以不会只剩黑屏。
- 继续保留：画中画完成提示、系统横幅映射、空回保护、选区长截图、液态 UI、自动防息屏、后台保活。

## 注意

iOS 不允许把完整可点击网页作为安卓式悬浮窗放到其他 App 上。画中画里的网页镜像只能看，不能直接点；后台时如果系统冻结 WKWebView，镜像可能停在最后一帧，但实时桥接文字会继续更新。

服务端插件不用更换，直接覆盖 GitHub 后重新构建 IPA。
