# 云洞酒馆 iOS v3.2

本版基于 v3.1 稳定版继续优化：

- 新增底部原生输入栏：可以直接在 App 底部输入并发送到 SillyTavern，减少 iPhone 17 Pro Max 圆角和手势条影响。
- 增强后台保活：进入酒馆后启用静音音频心跳、后台任务和防熄屏保活；画中画和生成中会继续保持。
- 悬浮球和展开面板改为液态透明玻璃风格。
- 设置页和切换云洞页改为玻璃卡片风格。
- 保留 v3.1 的空回误报保护、画中画完成提示、系统横幅映射、选区长截图。

## 使用

上传到 GitHub 后运行 Actions 构建 IPA。服务端插件不用更换。

进入酒馆后默认开启原生输入栏。如果不想用，可进入：

`悬浮球 → 设置 → 原生输入栏`

关闭后会恢复网页原来的底部发送框。

## 后台说明

iOS 不能像安卓一样永久后台运行任意网页，本版已经使用 App 端能做的最稳保活方式：静音音频心跳 + 画中画 + 短时后台任务 + 服务端实时桥接。长时间后台仍受系统调度影响。
