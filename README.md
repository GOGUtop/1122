# TavernSwitcher v3.9

本版重点修复键盘收回后立刻再次点击输入框时的卡顿。

- 键盘收回后增加 0.82 秒冷却保护。
- 冷却期间不恢复重型 JS / 图片扫描 / 液态面板。
- 再次点击输入框会取消恢复任务，避免主线程撞车。
- 保留 v3.8 的下载中心、极速模式、快速重 Roll、长截图和画中画文字提示。

# 云洞酒馆 iOS v3.8

本版重点处理 iOS 上酒馆卡顿、输入框卡顿、角色卡加载慢的问题。

## 新增/修改

- 去掉网页镜像画中画后的后续优化保留，只保留轻量实时文字画中画。
- 输入法弹出/收回期间不再重复注入性能脚本，减少键盘动画卡顿。
- 输入法收回方式改为更轻的 onDrag，避免 interactive 键盘持续拖动时触发大量回流。
- 键盘弹出时悬浮球完全隐藏，避免液态 UI 和键盘动画抢性能。
- 增加角色卡加载加速脚本：
  - 角色卡/头像/群组卡片启用 content-visibility。
  - 图片自动 lazy loading / async decoding。
  - offscreen 角色卡降低绘制负载。
- 极速模式进一步降低背景、阴影、毛玻璃、长聊天记录渲染负担。
- 保留下载中心、导出接管、快速重Roll、长截图、画中画完成提示、横幅映射。

## 建议设置

悬浮球 → 设置 → 丝滑性能 → 极速模式：保持开启。

如果还是卡，可以把页面比例调到 92%，底部安全距离保持 26–32px。
