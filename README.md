# TavernSwitcher v2.8 tools polish

本版基于 v2.7 继续优化，不需要更换服务端插件。

## 新增/调整

- 进入酒馆时悬浮球默认收起，并自动吸附屏幕边缘。
- 划词菜单更稳：选中文字后浮现「翻译 / 卡片」，减少乱闪和误消失。
- 划词翻译优化：
  - 增加本地缓存，重复翻译同一段会更快；
  - 中文内容自动识别，不再无意义请求网络；
  - 翻译面板分为原文/译文；
  - 增加复制译文按钮。
- 生成卡片优化：
  - 新增 8 套主题：星夜玻璃、奶油便签、赛博霓虹、樱粉胶片、墨绿诗页、极光蓝紫、黑金剧场、极简白卡；
  - 点「卡片」后先选择主题，再生成分享图。
- 保留 v2.6/v2.7 的画中画提示映射横幅逻辑。

## 安装

覆盖上传 GitHub 仓库，运行 Actions 构建 IPA。安装版本号应为 2.8（19）。
