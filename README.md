# TavernSwitcher v3.0 stable PiP banner

本版基于 v2.9 做稳定化处理：

- 删除不稳定的划词翻译、生成卡片和红色错误弹窗自动翻译。
- 保留稳定功能：系统画中画、画中画完成状态条、状态条映射顶部横幅、选区长截图、悬浮球工具。
- 画中画状态条映射出的系统横幅，现在会附带一次系统默认提示音 / 震动。
- 仍然只有画中画完成状态条可以触发横幅，酒馆原生事件、服务端桥接、后台轮询不会直接弹通知，减少重复提醒。
- “测试顶部横幅和震动”按钮已保留在悬浮球设置里。

覆盖上传 GitHub 仓库，运行 Actions 构建 IPA。安装版本号应为 3.0（21）。

服务端插件不用更换。
